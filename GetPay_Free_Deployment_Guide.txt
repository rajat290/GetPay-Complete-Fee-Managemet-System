## ðŸš€ STEP-BY-STEP FREE DEPLOYMENT

### Phase 1: Database Setup (MongoDB Atlas - Free)

#### 1.1 Create MongoDB Atlas Account
1. Go to [mongodb.com/atlas](https://www.mongodb.com/atlas)
2. Sign up for free account
3. Create new cluster (choose free tier)
4. Configure network access: Allow from anywhere (0.0.0.0/0)
5. Create database user with password

#### 1.2 Get Connection String
```bash
# Your connection string will look like:
mongodb+srv://username:password@cluster.mongodb.net/getpay?retryWrites=true&w=majority
```

### Phase 2: Backend Deployment (Render - Free)

#### 2.1 Prepare Backend for Deployment
```bash
# Update backend/package.json
{
  "scripts": {
    "start": "node server.js",
    "dev": "nodemon server.js"
  },
  "engines": {
    "node": "18.x"
  }
}
```

#### 2.2 Create Production Environment Variables
Create `backend/.env` file:
```bash
NODE_ENV=production
PORT=5000
MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/getpay?retryWrites=true&w=majority
JWT_SECRET=your-production-jwt-secret-key
RAZORPAY_KEY_ID=your-razorpay-key-id
RAZORPAY_KEY_SECRET=your-razorpay-key-secret
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
```

#### 2.3 Deploy to Render
1. Go to [render.com](https://render.com)
2. Sign up for free account
3. Create new Web Service
4. Connect your GitHub repository
5. Configure:
   - Environment: Node
   - Build Command: `npm install`
   - Start Command: `npm start`
   - Environment Variables: Add all from .env
6. Deploy!

#### 2.4 Get Backend URL
After deployment, you'll get:
```
https://getpay-backend.onrender.com
```

---

### Phase 3: Frontend Deployment (Vercel - Free)

#### 3.1 Update Frontend API URL
Update `frontend/src/services/api.js`:
```javascript
import axios from "axios";

const api = axios.create({
  baseURL: "https://getpay-backend.onrender.com/api", // Update this
});

api.interceptors.request.use((config) => {
  const token = localStorage.getItem("token");
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

export default api;
```

#### 3.2 Create Vercel Configuration
Create `frontend/vercel.json`:
```json
{
  "buildCommand": "npm run build",
  "outputDirectory": "dist",
  "devCommand": "npm run dev",
  "installCommand": "npm install"
}
```

#### 3.3 Deploy to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Import your repository
4. Configure:
   - Framework: Vite
   - Root Directory: frontend
   - Build Command: npm run build
   - Output Directory: dist
5. Deploy!

#### 3.4 Get Frontend URL
After deployment, you'll get:
```
https://getpay-frontend.vercel.app
```

## ðŸ†˜ TROUBLESHOOTING

### Common Issues & Solutions

#### 1. CORS Issues
```javascript
// Add this to backend/server.js
app.use(cors({
  origin: ['https://your-frontend.vercel.app'],
  credentials: true
}));
```

#### 2. Environment Variables Not Working
- Check spelling in variable names
- Ensure no quotes around values in Render/Vercel
- Restart services after changes

#### 3. MongoDB Connection Issues
- Verify connection string format
- Check IP whitelist in MongoDB Atlas
- Ensure database user has correct permissions

#### 4. Build Failures
- Check Node.js version compatibility
- Ensure all dependencies are in package.json
- Check for missing environment variables
